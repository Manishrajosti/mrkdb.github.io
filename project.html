Video face detection/openCV

It is the python app which detects human faces from a running video file or a running camera. OpenCV and Haarcascade are used to detect faces and make rectangle around the detected human faces.

Automated Website Blocker & Informer/Twilio API

It is the python app which blocks the website provided by the user and lets user to set time to start and end blocking time period. And once set, it will do automatically everyday on set time and texts/informs user to their's phone. Twilio API is used to send text to user. And changes are made it user's hosts file.

Automated Umbrella Remainder/Twilio API/OpenWeatherMap API

This python app gives the weather information like; temperature, wind, humidity, heat index and rain and
reminds you to get umbrella if it is raining or going to rain. It sends text sms to phone every set time and give weather info.
Twilio API and OpenWeatherMap API are used to send text and retrieve weather data.
